Tutorial
Step by Step
  1. Click start Game and wait for the tiles to appear on screen
  2. Presented to you is four black tiles which will flip over revealing a color.
  3. Your challenge is to remember the sequence in which the tiles appear

  4. If you remember the sequence correctly you can proceed.

  5. If you fail to remember the pattern that was displayed to you, you will be given the option to "preview again"
  6. If you choose strict mode the "preview again" option will not be available and you will return to level 0

  7. Try and reach level 20, Good Luck memorising!!
